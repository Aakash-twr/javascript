1. Object.keys(obj_name) - It will return us an array of strings with keys of object.

2. Object.keys(obj_name)[i] where i=0,1,2,3,etc - this will allow us to access the elements of array.

3. For-In loop is used to loop through the keys of objects.

4. String Interpolation - Variable inside string using template literals

5. Escape sequence characters - \ , \n etc used to escape any sequence in strings

6 . strings in javascript are immutable - Strings cannot be changed only copied then modified without affecting the original string.
